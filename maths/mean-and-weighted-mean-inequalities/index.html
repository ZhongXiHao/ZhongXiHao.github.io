Mean and Weighted Mean Inequalities

https://imomath.com/index.cgi?page=inequalitiesMeanMinkowskiHolder

Mean inequalities

众所周知对于$a_1,\ a_2,\ \cdots,\ a_n \in \mathbb{R}^+$， 算术平均数为：$A(a_1,a_2,\dots, a_n)=\dfrac{a_1+a_2+\cdots+a_n}n$，\ 几何平均数为：$G(a_1, a_2,\dots, a_n)=\sqrt[n]{a_1a_2\cdots a_n}$， 调和平均数为：$H(x_1,\dots, x_n)=\dfrac{n}{\dfrac1{x_1}+\cdots+\dfrac1{x_n}}$，\ 平方平均数为：$Q(x_1,\dots, x_n)=\sqrt{\dfrac{x_1^2+\cdots+x_n^2}n}$。 他们间的关系如下： $$Q(x_1,\dots, x_n)\geq A(x_1,\dots, x_n)\geq G(x_1,\dots, x_n)\geq H(x_1,\dots, x_n)$$ 等号成立条件为：$x_1=\dots=x_n$

下面，将使用一个函数概括上述四个不等式 $$ M_r(x)= \begin{cases}\left(\dfrac{x_1^r+x_2^r+\cdots+x_n^r}{n}\right)^{\dfrac{1}{r}} & \text { if } r \neq 0 \ \sqrt[n]{x_1 x_2 \cdots x_n} & \text { if } r=0\end{cases} $$

当$r=-1,\ 0,\ 1,\ 2$时分别为调和平均数，几何平均数，算术平均数，平方平均数

$\textbf{更一般地，对于任意$\mathbf{a\leq b}$，有$\mathbf{M_a(x) \leq M_b(x)}$}$

下证明当$r=0$时为几何平均数：

$$M_r(x) = \left(\dfrac{x_1^r+x_2^r+\cdots+x_n^r}{n}\right)^{\dfrac{1}{r}} = e^{\dfrac{1}{r}\ln \left(\dfrac{x_1^r+x_2^r+\cdots+x_n^r}{n} \right)}$$

由洛必达法则可知： $$ \begin{aligned} \lim _{r \rightarrow 0} \dfrac{1}{r}\ln \left(\dfrac{x_1^r+x_2^r+\cdots+x_n^r}{n} \right) &=\lim _{r \rightarrow 0} \dfrac{\dfrac{1}{n} \left( x_1^r \ln x_1+\cdots+x_n^r \ln a_n \right)}{\left(\dfrac{x_1^r+x_2^r+\cdots+x_n^r}{n} \right)} \ &=\dfrac{1}{n} \left( \ln x_1+\cdots+\ln x_n \right) \ &=\ln \left(x_1 \cdots x_n\right)^{\dfrac{1}{n}} . \end{aligned} $$

$\textbf{注：}$第一个等号需要用到复合函数求导

$\therefore r=0$时 $M_0(r) = \sqrt[n]{x_1 x_2 \cdots x_n}$

Weighted mean inequalities

现在考虑更为一般的加权情况，设$m_1,\ m_2,\ \cdots,\ m_n \in \textbf{R}^{+}$，且$m_1+m_2+\cdots +m_n = 1$ $$ M_r(x)= \begin{cases}\left(x_1^r m_1+x_2^r m_2+\cdots+x_n^r m_n\right)^{\dfrac{1}{r}} & \text { if } r \neq 0 \ x_1^{m_1} x_2^{m_2} \cdots x_n^{m_n} & \text { if } r=0\end{cases} $$

同样地，对于任意$a\leq b$，有$M_a(x) \leq M_b(x)$

而对于$r=0$的时候，证明类似于1。

Problems

Problem 1 (Young’s inequality)
设$a,\ b,\ p,\ q \in \mathbb{R}^+$，且$\dfrac{1}{p}+\dfrac{1}{q}=1$，求证：

$$\dfrac{a^p}p+\dfrac{b^q}q\geq ab$$

Problem 2
设$a,\ b,\ c, \in \mathbb{R}^+$，求证：

$$\dfrac{a^2+b^2+c^2}{a+b+c}\geq \left(a^a b^b c^c\right)^{\dfrac1{a+b+c}}$$

Problem 3
设$a,\ b,\ c, \in \mathbb{R}^+$，求证： $$\dfrac{(b+c)^a(c+a)^b(a+b)^c}{2^{a+b+c}}\leq \left(\dfrac{a+b+c}3\right)^{a+b+c}\leq a^ab^bc^c.$$
